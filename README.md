# FORGE Download Site

A lightweight landing page for downloading FORGE, my desktop AI engineering IDE.

This site has one focused purpose: give users a clean, simple place to learn what FORGE is and download the latest desktop build.

## Live Site

[FORGE Download Site](https://your-live-link-here.com)

## Related Project

[FORGE Desktop App](https://github.com/JoshElieson/PROMPT-ENG-v2.0)

## Why I Built It

FORGE is a desktop application, so it needed a separate public download page outside of the main app repository. I built this site to make the installation flow feel more polished and product-like, instead of requiring users to find a release buried inside GitHub.

## Features

- Clean landing page for FORGE
- Direct download call-to-action
- Short product explanation
- Responsive layout
- Lightweight frontend
- Hosted as a public download destination
- Designed to support future release/update information

## Tech Stack

- **Frontend:** React, TypeScript
- **Styling:** Tailwind CSS
- **Build Tool:** Vite / modern frontend tooling
- **Deployment:** Vercel

## Engineering Highlights

- Built a focused product landing page for a desktop software project
- Designed the site around a single clear conversion goal: downloading FORGE
- Created a lightweight deployment separate from the desktop app codebase
- Improved the user-facing distribution experience for a native application
- Kept the interface simple, fast, and easy to understand
